# 중복 없는 요소 찾아내기

`arr`은 배열입니다.

배열 내 유일한 요소를 찾아주는 함수 `unique(arr)`를 작성해보세요. 

예시:

```js
function unique(arr) {
  /* your code */
}

let strings = ["Hare", "Krishna", "Hare", "Krishna",
  "Krishna", "Krishna", "Hare", "Hare", ":-O"
];

alert( unique(strings) ); // Hare, Krishna, :-O
```

# pow(x,n) 함수 만들기

`x`의 `n`제곱을 반환해주는 함수, `pow(x,n)`를 만들어보세요. `x`의 `n` 제곱은 `x`를 `n`번 곱해서 만들 수 있습니다.

```js
pow(3, 2) = 3 * 3 = 9
pow(3, 3) = 3 * 3 * 3 = 27
pow(1, 100) = 1 * 1 * ...* 1 = 1
```

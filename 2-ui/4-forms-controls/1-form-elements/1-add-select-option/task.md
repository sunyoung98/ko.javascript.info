# select에 옵션 추가하기

`<select>` 가 있습니다. :

```html
<select id="genres">
  <option value="rock">Rock</option>
  <option value="blues" selected>Blues</option>
</select>
```

자바스크립트를 사용하여 :

1. 선택한 옵션의 값과 텍스트를 표시하세요.
2. 옵션을 추가하세요 : `<option value="classic">Classic</option>`.
3. 그것을 선택하세요.
